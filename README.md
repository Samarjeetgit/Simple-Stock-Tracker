# Simple Stock Tracker

This project is a beginner-friendly Python script that helps users track their stock investments. Users can input stock names and quantities, and the program calculates the total investment value. It can also save results for future reference.

## Features

- Add multiple stocks with quantity
- Uses a predefined stock price list or allows manual price entry
- Calculates total investment
- Optionally saves investment details in a text file
- Prevents invalid entries and guides the user

## Tech Stack

- Python 3
- Dictionaries for storing stock prices and quantities
- Input/output handling
- File handling (optional)

## How to Use

1. Ensure Python 3 is installed on your system.
2. Open PowerShell or Command Prompt and navigate to the project folder.
3. Run the script

Follow the prompts to enter stock names and quantities.
When done, type done to finish and see the summary of your investment.

Purpose

This project is designed for learning Python basics such as dictionaries, loops, input/output, and simple arithmetic operations. It simulates a real-world task of managing personal stock investments.
